// Why GHOST exists

Certs teach theory.
Breaches teach truth.

Security certifications have you memorize frameworks and answer multiple choice questions. CTF platforms give you toy vulnerabilities in a lab environment that bears no resemblance to a real network. Neither of them teach you what actually happened during the Colonial Pipeline attack — or what the defenders should have done differently.

GHOST was built by a red teamer who spent years analyzing real breach reports and watching security teams struggle with the exact techniques that were documented in public advisories. The gap isn't knowledge — it's muscle memory. GHOST builds it by making you reconstruct the actual commands, in the actual order, against the actual context.

// Red Track
Think like the attacker.

Reconstruct the exact commands attackers used in real breaches. Build the lateral movement chain. Execute the credential harvest. Understand the kill chain from the inside.

// Blue Track
Respond like a defender.

Build the detection and response commands that should have caught each attack. Write the SIEM queries. Run the endpoint containment. Learn what defenders missed and why.
