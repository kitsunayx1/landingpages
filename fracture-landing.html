// Common Fractures
What AI leaves behind
These aren't theoretical. We find these in almost every AI-generated app we scan.
Critical

Exposed API Keys

Stripe, Supabase, OpenAI keys hardcoded in frontend JavaScript bundles. One View Source away from theft.

Critical

No Row Level Security

Supabase tables wide open. Any authenticated user can read, modify, or delete every other user's data.

Critical

Broken Authentication

JWTs that never expire, password resets with no rate limiting, admin routes with no auth check.

High

SQL / NoSQL Injection

AI generates queries by concatenating user input. Classic injection vectors that automated tools catch instantly.

High

Cross-Site Scripting

User input rendered without sanitization. Stored XSS in comments, profiles, and anywhere users type.

Medium

Missing Security Headers

No CSP, no HSTS, no X-Frame-Options. The basics that every production app needs but AI never adds.
