Why We Built This

App submission is a
paper trail problem.

Every rejected build, every metadata revision, every privacy policy version — it's all context you need the next time a review goes sideways. ShipDeck keeps that history without putting it in someone else's database.

01 /
One window for the whole process

From first draft to App Store approval, every artifact lives in ShipDeck. Checklists, screenshots, metadata, review history — managed together, not scattered across Notion, Finder, and your inbox.

02 /
Nothing uploaded anywhere

No login, no sync, no data leaves your Mac. Your app metadata, review notes, and build logs are stored locally in a SQLite database you own entirely.

03 /
Structured review history

Every rejection, resubmission, and resolution is logged with timestamps and notes. When Apple rejects the same guideline twice, you'll have the full thread to reference.

04 /
Privacy policy manager built in

Generate, version, and link your app's privacy policies directly from ShipDeck. No more hunting for which policy version was live when you submitted build 47.

Features

Everything the App Store
process demands.

📋
Submission Checklist

Pre-submission checklists for iOS and Android that evolve as store requirements change. Check off each item, track completion per build.

📸
Screenshot Manager

Organize screenshots by device size and locale. Track which sets are uploaded, approved, or pending for each submission.

🔒
Privacy Policy Manager

Version your privacy policies alongside your builds. Know exactly which policy was live at the time of every submission.

📝
Metadata Editor

Edit and track app name, subtitle, description, keywords, and promotional text. Diff between versions to see exactly what changed.

🔍
Review Log

Log every review event — approved, rejected, pending — with the reviewer's notes and your response. Full timestamped history per app.

📦
Build Tracker

Track build numbers, upload dates, test notes, and which store submission each build corresponds to. Never lose the thread between builds.

100% Local · macOS
Native Swift. No Electron. No cloud account required.

ShipDeck is built with Swift and SwiftUI for macOS. It uses SwiftData on-device — not a cloud sync layer. Open it, start tracking, close it. No API keys, no OAuth, no permissions dialogs asking to phone home.
