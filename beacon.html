Detection Capabilities

Six layers of visibility into your network.

BEACON layers network behavior, device inventory, AI file monitoring, and vulnerability scanning into a single console — with mobile alerts when something needs your attention.

🌐

Network Behavior Monitor

Builds behavioral baselines for every device on your network. Alerts when a device makes unexpected connections, changes communication patterns, or beacons unexpectedly.

🤖

AI Skill File Watcher

Monitors AI agent configuration files (Claude, Continue, Cursor) for prompt injection attempts, unauthorized permission escalation, and hash tampering. New threat, new detection.

📱

Mobile App (iOS + Android)

Companion app delivers real-time alerts, device inventory, and threat summaries to your phone. No cloud relay — connects directly to your BEACON instance over your local network.

🔍

IoT Device Inventory

Automatically discovers and classifies every device on your network — smart TVs, thermostats, cameras. Know what's connected before something else figures it out first.

🛡

CVE Exposure Scanner

Matches discovered devices and software against the NVD CVE database. Surfaces known vulnerabilities in your specific device models and firmware versions.

📡

Local AI Inference

Behavioral analysis and anomaly detection run entirely on-device. No API keys required. No traffic metadata sent offsite. Upgrade with your own cloud key for enhanced detection.
