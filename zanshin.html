// why_we_built_this

残心 — Remaining mind.

Zanshin is the state of focused awareness that persists after action. It's the awareness your terminal has never had — until now.

01

Context where you need it

The left panel shows your current working context, relevant skill suggestions, and knowledge graph hits from your Obsidian vault. All cwd-aware, all up to date.

02

Terminal that actually captures sessions

Every session is captured in asciinema v2 format with FTS5 command indexing. Search anything you've ever run — by command, by output, by working directory.

03

Themes that fit your environment

Load .toml theme files from ~/.zanshin/themes/. Sensitive-path auto-theming switches to a danger theme for remote and cloud sessions automatically.

04

Shell integration that goes deep

OSC 7 (cwd tracking), OSC 133 (command demarcation), preexec hooks. Zanshin knows where you are, what you just ran, and what you're about to type.

// platform_features

Everything in place.
Nothing in the way.

Designed for developers who live in the terminal and expect their tools to keep up.

🪟

Three-Panel Layout

Context panel (left) + SwiftTerm PTY (center) + ambient status (right). All panels resizable. Panels can be hidden per-session via keybind.

🎯

Skill Suggestions

LLM-powered suggestions based on your current cwd, recent commands, and Obsidian vault context. Relevant, not generic. Dismissed with Escape.

🔍

Session Search (FTS5)

Full-text search across every command you've ever run, with session context. Powered by SQLite FTS5. ⌘F opens in-pane, ⌘⇧F opens global search.

📚

Obsidian Integration

Point Zanshin at your vault root. Knowledge graph hits surface in the context panel when your cwd matches a note's local_path frontmatter.

🎨

Custom Theme System

TOML-based themes for full color control. Ships with five built-in themes. Auto-theming rules trigger on path patterns — protect yourself from accidents.

⌘

Pane Splitting

⌘D for vertical split, ⌘⇧D for horizontal. Each pane is an independent PTY session. Synchronized input mode for parallel command execution.
